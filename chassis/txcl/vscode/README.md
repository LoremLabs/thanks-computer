# txcl

Syntax highlighting and snippets for `.txcl` rule files — the declarative
event-routing DSL used by [thanks, computer.](https://www.thanks.computer).

## What you get

- Highlighting for keywords (`WHEN`, `SELECT`, `SET`, `EXEC`, `EMIT`, …),
  strings, regex literals, branch paths (`.x.y.z`, `@x`), the `&fn`
  function-call sigil, numbers, operators, and `#` comments — colors
  come from your active theme.
- `#` line-comment toggling (Cmd+/), bracket matching, auto-pairs.
- Snippets: `when`, `set`, `exec`, `emit`, full `rule` skeleton.
- File icon for `.txcl` in the Explorer.
